Programação IV

Ementa

Linguagem de Programação para a Web. Conceitos Básicos de Servidor Web. Construção de Interfaces visuais para a Web. Envio e recebimento de dados na Web. Persistência de Dados. Conceitos básicos de segurança na WEB. Sessões e Cookies. Arquitetura MVC.

OBJETIVO GERAL

Capacitar o aluno a elaborar sistemas com acesso via Web.
