LPV - Linguagem de Programação V

EMENTA

Utilização de frameworks de desenvolvimento para WEB com arquitetura MVC. Framework de persistência. Mapeamento Objeto-relacional. Introdução ao uso e desenvolvimento de Web Services.

OBJETIVO GERAL

Possibilitar ao aluno utilizar bibliotecas e técnicas que permitam uma maior produtividade no desenvolvimento de software.
